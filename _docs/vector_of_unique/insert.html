std::pair<const_iterator, bool> insert( const_iterator pos, const T& value );
(1)
std::pair<const_iterator, bool> insert( const_iterator pos, T&& value );
(2)
template< class input_it >
const_iterator insert( const_iterator pos, input_it first, input_it last );
(3)
const_iterator insert( const_iterator pos, std::initializer_list<T> ilist );
(4)

Inserts elements at the specified location in the container.

1) Inserts a copy of value before pos.
If any of the following conditions is satisfied, the behavior is undefined:
  • T is not CopyInsertable into vector_of_unique.
  • T is not CopyAssignable.
2) Inserts value before pos, possibly using move semantics.
If any of the following conditions is satisfied, the behavior is undefined:
  • T is not MoveInsertable into vector_of_unique.
  • T is not MoveAssignable.
3)4)3) Inserts elements from range [first, last) before pos.

This overload participates in overload resolution only if InputIt satisfies the requirements of LegacyInputIterator.

If any of the following conditions is satisfied, the behavior is undefined:
  • T is not EmplaceConstructible into vector_of_unique from *first.
  • T is not MoveInsertable into vector_of_unique.
  • T does not satisfy the requirements of MoveConstructible, MoveAssignable or Swappable.
  • first or last are iterators into *this.
4) Inserts elements from initializer list ilist before pos.
Equivalent to insert(pos, ilist.begin(), ilist.end()).

All iterators (including the cend() iterator) are invalidated. References are invalidated too, unless pos == cbegin() or pos == cend(), in which case they are not invalidated.

Parameters

pos - iterator before which the content will be inserted (pos may be the cend() iterator)
value - element value to insert
first, last - the pair of iterators defining the range of elements to insert
ilist - std::initializer_list to insert the values from

Return value

1,2) A pair consisting of an iterator to the inserted element (or pos) and a bool value set to true if and only if the insertion took place.
3) An iterator to the first element inserted, or pos if first == last.
4) An iterator to the first element inserted, or pos if ilist.

Complexity

1,2)
Best case: If the element is a duplicate, the complexity is O(1).
Average case: If reallocation happens, linear in the number of elements of the vector after insertion; otherwise, linear in the number of elements inserted plus std::distance(pos, end()) if any element is inserted.
Worst case: linear in the size of the container.
3)
Best case: If the elements are all duplicates, the complexity is O(N).
Average case: between O(N) and O(N * M), where N is std::distance(first, last).
Worst case: O(N * M), where M is the size of original vector_of_unique.
4)
Best case: If the elements are all duplicates, the complexity is O(N).
Average case: between O(N) and O(N * M) , where N is the size of ilist.
Worst case: O(N * M).

Exceptions

If an exception is thrown other than by

  • the copy constructor of T,
  • the move constructor of T,
  • the copy assignment operator of T,
  • the move assignment operator of T,
  • any InputIt operation,

this function has no effect (strong exception safety guarantee).

If an exception is thrown when inserting a single element at the end, and T is CopyInsertable into *this or std::is_nothrow_move_constructible<T>::value is true, this function has no effect (strong exception safety guarantee). Otherwise, if an exception is thrown by the move constructor of a non-CopyInsertable T, the effects are unspecified.
