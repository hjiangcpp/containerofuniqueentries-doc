void pop_back();

Removes the last element of the container.

If the container is empty, the call has no effect.

Iterators and references to the last element are invalidated. The cend() iterator is also invalidated.

Parameters

(none)

Return value

(none)

Complexity

Average case: constant.
Worst case: linear in the size of the container.

Exceptions

Throws nothing.
