std::pair<const_iterator, const_iterator> equal_range( const Key& key ) const;
(1)
template< class K >
std::pair<const_iterator, const_iterator> equal_range( const K& x ) const;
(2) (since C++20)
1) Returns a range containing all elements with key equivalent to key. Since all elements in the container are unique, the returned range contains at most one element.
2) Returns a range containing all elements with key that compares equivalent to the value x. This overload participates in overload resolution only if Hash::is_transparent and KeyEqual::is_transparent are valid and each denotes a type. This assumes that such Hash is callable with both K and Key type, and that the KeyEqual is transparent, which, together, allows calling this function without constructing an instance of Key.

Parameters

key - key value of the elements to search for
x - a value of any type that can be transparently compared with a key

Return value

A std::pair of const_iterators. Since all elements are unique, the range is either {it, it+1} if an element with the given key is found, or {cend(), cend()} if not found.

Complexity

O(N), where N is the size of the container.
