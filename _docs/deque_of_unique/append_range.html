template< class R >
void append_range(R&& rng);
(1) (since C++23)

Appends unique elements from the range rng to the end of the container. Elements already present in the container and intra-range duplicates are silently skipped.

Only available when __cplusplus >= 202302L.

All iterators (including the cend() iterator) are invalidated. No references are invalidated.

Parameters

rng - a range of elements to append

Return value

(none)

Complexity

Average case: linear in the size of rng. Worst case: quadratic.
