template< class InputIt >
void assign( InputIt first, InputIt last );
(1)
void assign( std::initializer_list<T> ilist );
(2)

Replaces the contents of the container. Only unique elements are stored.

1) Replaces the contents with copies of those in the range [first, last). Duplicate elements are ignored.
If either argument is an iterator into *this, the behavior is undefined.

This overload participates in overload resolution only if InputIt satisfies LegacyInputIterator.

2) Replaces the contents with the elements from ilist. Duplicate elements are ignored.

All iterators (including the cend() iterator) and all references to the elements are invalidated.

Parameters

first, last - the range to copy the elements from
ilist - std::initializer_list to copy the values from
Type requirements
-
T must meet the requirements of CopyInsertable in order to use overloads (1, 2).

Return value

(none)

Complexity

1)
Average case: O(M + N), where N is std::distance(first, last) and M is the number of elements in the original container.
Worst case: O(M + N2).
2)
Average case: O(M + N), where N is std::size(ilist) and M is the number of elements in the original container.
Worst case: O(M + N2).

Exceptions

If an exception is thrown, the container is left in a valid but unspecified state.
