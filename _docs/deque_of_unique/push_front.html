bool push_front(const T& value);
(1)
bool push_front(T&& value);
(2)

Prepends the given element value to the beginning of the container if it is not already present. This ensures that all entries in the container remain unique.

1) The new element is initialized as a copy of value.
2) value is moved into the new element.

All iterators (including the cend() iterator) are invalidated. No references are invalidated.

Parameters

value - the value of the element to prepend
Type Requirements
- T must meet the requirements of CopyInsertable in order to use overload (1).
- T must meet the requirements of MoveInsertable in order to use overload (2).

Return value

true if the insertion is successful; otherwise, false if the element already exists in the container.

Complexity

Constant.

Exceptions

If an exception is thrown for any reason, this function has no effect (strong exception safety guarantee).
