template< class R >
iterator insert_range(const_iterator pos, R&& rng);
(1) (since C++23)

Inserts unique elements from the range rng before pos. Elements already present in the container and intra-range duplicates are silently skipped.

Only available when __cplusplus >= 202302L.

All iterators (including the cend() iterator) are invalidated. References are invalidated too, unless pos == cbegin() or pos == cend(), in which case they are not invalidated.

Parameters

pos - iterator before which the new elements will be inserted
rng - a range of elements to insert

Return value

Iterator pointing to the first inserted element, or pos if no elements were inserted.

Complexity

Average case: linear in the size of rng. Worst case: quadratic.
